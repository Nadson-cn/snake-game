
# Snake Game

O famoso jogo da cobrinha em ReactJS, utilizando HTMLCanvasElement, react hooks e audio. Simples joguinho para treinar os conhecimentos.




## Instalação

Instale snake-game com yarn

```bash
  yarn install
  yarn dev
```
